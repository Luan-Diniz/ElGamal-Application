# Front-end feito em React

Para executar é necessário instalar Node.js e Vite

## Executar

Antes da primeira execução é necessário:
```shell
npm install
```

Para rodar no port default:
```shell
npm run dev
```

Para rodar em um port específico definido em _package.json_:
```shell
npm run dev:port1
```

Ou
```shell
npm run dev:port2
```

Ou
```shell
npm run dev:port3
```
